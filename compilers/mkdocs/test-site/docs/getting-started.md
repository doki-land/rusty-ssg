# Getting Started

This is a documentation page for getting started with the test site.

## Installation

1. Install the mkdocs compiler
2. Create a new project
3. Configure the site
4. Add content
5. Build the site

## Configuration

The site is configured using the `mkdocs.yml` file. Here's an example configuration:

```yaml
site_name: Test Site
theme:
  name: material
plugins:
  - search
```

## Deployment

To deploy the site:

1. Run `mkdocs build`
2. Upload the `site` directory to your hosting provider
