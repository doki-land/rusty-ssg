# Getting Started

## Installation

```bash
# Install MkDocs
pip install mkdocs mkdocs-material

# Create your first document
mkdir -p docs
```

## Basic Usage

1. Create a `docs` directory
2. Add your Markdown files
3. Configure `mkdocs.yml`
4. Run the dev server

```bash
# Run dev server
mkdocs serve

# Build for production
mkdocs build
```
